능력단위 평가 — 2번 문제

[2번 문제] 컨트롤러로 물체 잡기.
학습 목표: Near-Far Interactor를 컨트롤러에 부착하고, XR Grab Interactable로 잡을 수 있는 물체(Cube)를 구성하여 컨트롤러로 물체를 집을 수 있다.
사전 조건 Sol_01 씬이 완성되어 있어야 합니다 (좌·우 컨트롤러가 기기를 따라 움직이는 상태).

Sol_02 — 풀이

씬 준비

  1. 씬 복사 — Sol_01 씬을 복사해서 Sol_02 로 만든다.
  2. 씬 열기 — Sol_02 씬을 열어준다.

컨트롤러에 인터랙터 부착

  1. Left Controller 펼치기 — Hierarchy에서 Left Controller를 펼쳐준다.
  2. 인터랙터 배치 — Project 창에서 near 로 검색하면 Left_NearFarInteractor 가 나온다. 끌어다가 Left Controller 의 자식으로 넣어준다.
  3. 위치 초기화 — Left_NearFarInteractor 의 Transform 위치를 초기화한다.

잡을 물체(Cube) 만들기

  1. Cube 생성 — Hierarchy에서 빈 오브젝트를 생성하고 이름을 Cube 로 변경한다.
  2. 위치 설정 — Cube 의 위치를 초기화한 뒤 (0, 1.1, 0.5) 로 설정한다.
  3. Visuals 생성 — Cube 의 자식으로 빈 오브젝트를 생성하고 이름을 Visuals 로 변경한다.
  4. Mesh 생성 — Visuals 를 선택하고 우클릭 → 3D Object > Cube 를 선택한다. 생성된 오브젝트의 이름을 Mesh 로 변경하고, 거기에 붙은 Box Collider 를 제거한다.
  5. Col 생성 — Cube 를 선택하고 우클릭해서 빈 오브젝트를 생성한 후 이름을 Col 이라고 한다.
  6. 충돌체 부착 — Col 오브젝트를 선택하고 Add Component 버튼을 눌러 Box Collider 를 부착한다.
  7. 크기 설정 — Cube(최상위 오브젝트)를 선택하고 Scale을 (0.1, 0.1, 0.1) 로 수정한다.

잡기(Grab) 설정

  1. XR Grab Interactable 부착 — Cube 를 선택하고 XR Grab Interactable 컴포넌트를 부착한다.
  2. 중력 끄기 — Rigidbody 가 자동으로 부착되었을 텐데, Use Gravity 를 체크 해제한다.
  3. 던지기 끄기 — XR Grab Interactable 컴포넌트의 Throw On Detach 를 체크 해제한다.

확인 & 오른손 적용

  1. 동작 확인 — 실행(Play) 후 왼손으로 Cube를 잡아본다.
  2. 오른손 적용 — 같은 방식으로 오른손에도 적용한다. Project에서 near 검색 → Right_NearFarInteractor 를 Right Controller 자식으로 넣고 위치 초기화.
완성 Cube 구조 
Cube                 (Scale 0.1 / XR Grab Interactable + Rigidbody)
├─ Visuals
│  └─ Mesh           (3D Cube, Box Collider 제거 — 보이기용)
└─ Col               (Box Collider — 충돌/잡기용)
보이는 메쉬(Mesh)와 충돌체(Col)를 분리해, 시각 표현과 물리 충돌을 따로 관리합니다.
