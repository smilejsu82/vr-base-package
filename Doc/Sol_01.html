능력단위 평가 — 1번 문제

[1번 문제] 새 씬을 만들어서 왼손·오른손 컨트롤러 모델이 기기의 컨트롤러 움직임에 따라 함께 움직이는지 확인하기.
학습 목표: XR Origin (VR) 구성, 컨트롤러 비주얼 배치, Tracked Pose Driver (Input System)를 이용해 컨트롤러 모델이 실제 기기 입력을 따라 추적되도록 설정할 수 있다.

Sol_01 — 풀이

왼손 컨트롤러 만들기

  1. 메인 카메라 삭제 — Hierarchy의 기본 Main Camera를 삭제한다.
  2. XR Origin (VR) 생성 — Hierarchy > 우클릭 > XR > XR Origin (VR) 를 생성한다.
  3. 위치 초기화 & 카메라 오프셋 설정
    • XR Origin (VR)의 Transform 위치를 초기화한다 (Position 0, 0, 0).
    • XR Origin 컴포넌트의 Camera Y Offset 값을 0 으로 설정한다.
    • Camera Offset 게임오브젝트의 Transform 위치도 초기화한다.
  4. Left Controller 생성 — XR Origin을 선택한 상태에서 빈 오브젝트를 생성하고 이름을 Left Controller 로 변경한다.
  5. Visuals 생성 — Left Controller를 선택하고 우클릭해서 빈 오브젝트를 생성하고 이름을 Visuals 로 변경한다.
  6. 컨트롤러 모델 검색 — Project 창에서 xr controller left 를 검색한다.
  7. 모델 배치 — 검색된 xr controller left 를 Visuals 의 자식으로 넣는다.
  8. 위치 초기화 — 방금 넣은 모델의 Transform 위치를 초기화한다.
  9. Tracked Pose Driver 부착 — Left Controller를 선택하고 Tracked Pose Driver (Input System) 컴포넌트를 부착한다.
  10. 설정 도구 열기 — 메뉴 Tools > VR > Tracked Pose Driver Setup 을 선택한다.
  11. 자동 할당 — Add & Assign Tracked Pose Driver 버튼을 눌러준다.
  12. 설정 확인 — 콘솔 로그 [Pose Driver Setup] 1개의 컨트롤러에 Tracked Pose Driver를 설정했습니다. 를 확인한다. Left Controller 게임오브젝트를 선택해 인스펙터를 보면 Position Input, Rotation Input, Tracking State Input 이 채워진 것을 볼 수 있다.
  13. 동작 확인 — 실행(Play) 후 왼손 컨트롤러를 움직여 모델이 함께 움직이는지 확인한다.

오른손 컨트롤러 만들기

  1. 오른손도 동일하게 제작 — 위 4~13단계를 그대로 반복하되 다음만 바꾼다.
    • 빈 오브젝트 이름: Right Controller
    • Project 검색어: xr controller right
    • 나머지(Visuals 생성, 모델 배치, 위치 초기화, Tracked Pose Driver 부착, 자동 할당)는 동일하게 진행
완성 하이어라키 
XR Origin (VR)
└─ Camera Offset
   ├─ Main Camera
   ├─ Left Controller        (Tracked Pose Driver)
   │  └─ Visuals
   │     └─ XR Controller Left (모델)
   └─ Right Controller       (Tracked Pose Driver)
      └─ Visuals
         └─ XR Controller Right (모델)
참고 컨트롤러 모델(XR Controller Left/Right)은 XRI Starter Assets 샘플에 포함되어 있습니다. 검색이 안 되면 Package Manager에서 XR Interaction Toolkit ▸ Samples ▸ Starter Assets 를 먼저 Import 하세요.
확인 포인트 실제 기기(Quest/Link)를 연결한 상태에서 Play 해야 컨트롤러 추적이 동작합니다. 좌·우 컨트롤러를 움직였을 때 두 모델이 각각 따라 움직이면 통과입니다.
