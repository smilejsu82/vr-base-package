능력단위 평가 — 3번 문제

[3번 문제] 컨트롤러 모델 대신 손 모델을 표시하고, 그립 버튼을 누르면 손이 쥐는 애니메이션이 동작하도록 만들기. (큐브도 잡을 수 있어야 함)
학습 목표: 손 모델 교체, HandGlow 머티리얼 적용, Hand 컴포넌트의 Input Device Characteristics 설정, Animator + XRHand 컨트롤러로 그립 애니메이션을 연결할 수 있다.
사전 조건 Sol_02 씬이 완성되어 있어야 합니다 (컨트롤러로 큐브를 잡는 상태).

Sol_03 — 풀이

씬 준비 & 손 모델 교체

  1. 씬 복사 — Sol_02 씬을 복사해서 Sol_03 으로 만든다.
  2. 씬 열기 — Sol_03 씬을 열어준다.
  3. 컨트롤러 모델 제거 — Left Controller 의 Visuals 자식에 있는 XR Controller Left 를 제거한다.
  4. 손 모델 배치 — XR Hands > 1.7.3 > HandVisualizer > Models 에 있는 LeftHand 를 끌어다가 Visuals 에 넣는다.
  5. 위치 초기화 — 방금 넣은 모델의 Transform 위치를 초기화한다.
  6. 머티리얼 적용 — Packages > VR Base Package 에서 HandGlow 머티리얼을 끌어다가 Visuals 자식에 있는 LeftHand/LeftHand 에 넣는다.

Hand & Animator 설정

  1. Hand 컴포넌트 — 최상위 LeftHand 를 선택하고 Add Component 로 Hand 컴포넌트를 부착한다.
  2. 입력 특성 설정 — Hand 컴포넌트의 Input Device Characteristics 를 선택해 Controller, Left 둘 다 선택한다.
  3. Animator 부착 — LeftHand 를 선택해 Add Component 로 Animator 컴포넌트를 부착한다.
  4. 컨트롤러 할당 — VR Base Package에 있는 XRHand_L 애니메이터 컨트롤러를 Animator 컴포넌트의 Controller 에 할당한다.
  5. 1차 확인 — 실행 후 왼쪽 컨트롤러의 그립 버튼을 눌러 애니메이션이 동작하는지 확인한다.

손 각도 보정 (실행 중 → 복사/붙여넣기)

  1. 각도 수정 & 복사 — 동작은 하는데 손의 각도가 맞지 않는다면, 실행하는 도중에 LeftHand 의 Transform Rotation 을 바꿔가며 올바른 각도로 맞춘다. 예시 값: (-27.4, 5.98, 82.79)
    맞췄다면 Transform 컴포넌트에 우클릭 → Copy > Component 를 누른 후 에디터를 중지한다.
    주의실행(Play) 중에 바꾼 값은 정지하면 사라집니다. 반드시 정지 전에 Copy 해두세요.
  2. 붙여넣기 — (정지 후) LeftHand 를 선택하고 Transform에 우클릭 → Paste 로 붙여넣기 한다.
  3. 최종 확인 — 다시 실행해서 ① 손이 올바르게 나오는지 ② 그립 버튼을 누르면 애니메이션이 동작하는지 ③ 큐브를 잡을 수 있는지 확인한다.

오른손 적용

  1. 오른손도 동일하게 — 위 과정을 오른손에 적용한다.
    • Right Controller 의 XR Controller Right 제거
    • HandVisualizer > Models 의 RightHand 배치 → 위치 초기화 → HandGlow 적용
    • Hand 컴포넌트: Input Device Characteristics = Controller, Right
    • Animator 컨트롤러: XRHand_R 할당
    • 각도 보정 후 큐브 잡기 확인
