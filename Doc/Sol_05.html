능력단위 평가 — 5번 문제

[5번 문제] 실린더를 양손(왼손·오른손)으로 잡기. 각 손은 자기 포즈로 잡혀야 하고, 왼손→오른손 / 오른손→왼손 어느 순서로 잡아도 고스트 핸드 위치에 정확히 잡혀야 한다.
학습 목표: 양손용 콜라이더를 구성하고, 좌·우 핸드포즈를 각각 녹화하여 XR Grab Interactable의 다중(Multiple) 선택 모드 + Secondary Attach Transform 으로 두 손 잡기를 구현할 수 있다. 또한 잡을 물체용 컴포넌트(포즈 2개 보유)를 작성할 수 있다.
사전 조건 Sol_04 의 흐름(핸드포즈 녹화 → Grab Attach / Interactor Attach)을 이해하고 있어야 합니다.

Sol_05 — 풀이

1) 씬 준비 & 실린더 만들기

  1. 씬 복사 — Sol_04 를 복사해서 Sol_05 를 만든다.
  2. 씬 열기 — Sol_05 씬을 연다.
  3. Cube 삭제 — 기존 Cube 를 삭제한다.
  4. Cylinder 생성 — 빈 게임 오브젝트를 생성하고 이름을 Cylinder 로 변경한다.
  5. 위치 설정 — 위치를 (0, 1.1, 0.5) 로 변경한다.
  6. Visuals 생성 — Cylinder 를 선택하고 자식으로 Visuals 빈 오브젝트를 생성한다.
  7. Mesh 생성 — Visuals 아래 3D Object > Cylinder 를 생성한다.
  8. 크기 설정 — 크기를 (0.1, 0.3, 0.1) 로 변경한다.
  9. 콜라이더 제거 — 자동으로 붙은 Capsule Collider 를 제거한다.
  10. 이름 변경 — 이름을 Mesh 로 변경한다.
  11. Col 생성 — Cylinder 를 선택하고 자식으로 Col 빈 오브젝트를 만들고 Capsule Collider 를 추가한다.
  12. 콜라이더 사이즈 — Capsule Collider의 Radius = 0.05, Height = 0.2 로 변경한다.
  13. 좌·우 콜라이더 — Col 의 이름을 Col_L 로 변경한 후 복사해서 Col_R 를 생성한다.
  14. 위치 배치 — Col_L 은 위쪽으로, Col_R 은 아래쪽으로 위치시킨다. (양손이 잡을 두 지점)

2) 양손 핸드포즈 녹화

  1. 모드 전환 — Left/Right Controller 를 비활성화하고 Hand Visualizer 를 활성화한다.
  2. 좌·우 스냅샷 — 실행 후 왼손·오른손 핸드트래킹으로 손모양을 연출해 Tools > Hand Pose Recorder 로 Attach To Target 한다.
    • 왼손/오른손을 따로 만들어야 하므로 Hand Pose Recorder에서 Hand Name 을 변경한다(좌/우 구분).
    • Attach Target = 씬의 Cylinder
    • Ghost Hand Model = LeftHand / RightHand 프리팹(각각)
  3. 고스트 핸드 배치 — 플레이를 중지하고, 방금 스냅샷한 왼손·오른손 고스트 핸드를 Cylinder 에 넣어준다.

3) Grab / 그립 설정

  1. XR Grab Interactable — Cylinder 를 선택하고 XR Grab Interactable 컴포넌트를 부착한다.
  2. 물리 설정 — Rigidbody 의 Use Gravity 체크 해제, XR Grab Interactable 의 Throw On Detach 체크 해제.
  3. 모드 전환 — Hand Visualizer 를 비활성화하고 왼손·오른손 컨트롤러를 활성화한다.
  4. 그립 포즈 컴포넌트 — 왼손·오른손 컨트롤러에 Controller Grab Hand Pose 컴포넌트가 붙어 있어야 한다.
  5. Interactor Attach — Left Controller 를 선택하고 Tools > VR > Align Interactor Attach... 로 Attach Transform을 만든다. (오른쪽도 동일)
  6. Grab Attach — Cylinder 의 왼손 HandPose 스냅샷을 선택하고 Tools > VR > Create Grab Attach... 로 Attach Transform을 만든다. (오른쪽도 동일)

4) 양손(다중) 잡기 연결

  1. 다중 선택 모드 — Cylinder 를 선택하고 XR Grab Interactable 의 Select Mode 를 Single → Multiple 로 변경한다.
  2. Attach Transform 할당 — Cylinder 의 XR Grab Interactable의 Attach Transform 과 Secondary Attach Transform 에 각각 할당한다.
    • 왼손 → 왼손 Grab Attach (Ghost Wrist)
    • 오른손 → 오른손 Grab Attach (Ghost Wrist)

5) 실린더 컴포넌트 (포즈 2개) 작성

  1. Cylinder 컴포넌트 만들기 — (Cursor 등으로) Cube 컴포넌트와 같은 방식의 Cylinder 컴포넌트를 만들어 Cylinder 게임 오브젝트에 부착한다.
    주의 — 핸드포즈가 1개가 아니다
    • Assets/HandPoses 의 HandPose_Snapshot_Left 와 HandPose_Snapshot_Right 를 각각 할당할 수 있어야 한다.
    • Left Controller, Right Controller 의 Controller Grab Hand Pose 가 그에 맞는 포즈를 취해야 한다(왼손은 왼손 포즈, 오른손은 오른손 포즈).

6) 마무리 & 확인

  1. 잡았을 때 처리 — 잡는 순간, 왼손·오른손 컨트롤러 모델이 물리 컨트롤러를 움직여도 함께 움직이지 않아야 하며(포즈 고정), 고스트 핸드는 비활성화되어야 한다.
  2. 확인 포인트
    • 왼손 먼저 잡고 → 오른손 잡기
    • 오른손 먼저 잡고 → 왼손 잡기
    • 두 경우 모두 각 손이 고스트 핸드 위치(스냅샷 포즈) 에 정확히 잡혀야 한다.
