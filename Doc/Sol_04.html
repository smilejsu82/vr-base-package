능력단위 평가 — 4번 문제

[4번 문제] 핸드트래킹으로 큐브 잡는 손모양을 녹화(스냅샷) 하고, 그 포즈를 컨트롤러 잡기에 적용해서 컨트롤러로 큐브를 잡았을 때 손이 스냅샷한 그립 모양으로 잡히게 만들기.
학습 목표: Hand Visualizer로 핸드트래킹을 띄우고, Hand Pose Recorder로 그립 포즈를 스냅샷한 뒤 Grab Attach / Interactor Attach 도구로 컨트롤러 잡기에 손 포즈를 정렬할 수 있다.
사전 조건 Sol_03 씬이 완성되어 있어야 합니다. 핸드트래킹은 실제 기기(Quest/Link)에서만 동작합니다.

Sol_04 — 풀이

1) 씬 준비

  1. 씬 복사 — Sol_03 씬을 복사해서 Sol_04 로 만든다.
  2. 씬 열기 — Sol_04 를 열어준다.
  3. 컨트롤러 비활성화 — 씬에 있는 Left Controller, Right Controller 를 비활성화한다.

2) Hand Visualizer 구성 (핸드트래킹)

  1. 샘플 씬 열기 — Samples > XR Hands 밑에 있는 HandVisualizer 씬을 연다.
  2. 컴포넌트 복사 — XR Origin 밑에 있는 Hand Visualizer 를 선택하고, Hand Visualizer 컴포넌트에 우클릭 → Copy Component 로 복사한다.
  3. 빈 오브젝트 생성 — 다시 Sol_04 씬을 열고, XR Origin 에 우클릭해서 빈 오브젝트를 생성하고 이름을 Hand Visualizer 로 변경한다.
  4. 컴포넌트 붙여넣기 — 그 오브젝트의 Transform 에 우클릭 → 아까 복사한 컴포넌트를 Paste Component As New 로 붙여 넣는다.
  5. Hand Processor 부착 — Hand Visualizer 를 선택해 Add Component 로 Hand Processor 컴포넌트를 부착한다.
  6. 핸드트래킹 확인 — 실행해서 핸드트래킹이 잘 동작하는지 확인한다.

3) 핸드포즈 녹화 (Hand Pose Recorder)

  1. 도구 열기 — Tools > Hand Pose Recorder 를 열어준다.
  2. Attach Target — Attach Target 에 Cube 를 할당한다.
  3. Ghost Hand Model — Project 창의 XR Hands > 1.7.3 밑에 있는 LeftHand 프리팹을 Ghost Hand Model 에 넣어준다.
  4. 1차 스냅샷 — 실행해서 큐브를 잡는 포즈를 핸드트래킹으로 연출한 후, Hand Pose Recorder의 Attach To Target (즉시) 버튼을 눌러준다.
중요 — 큐브 크기 점검 (스냅샷 전에 반드시 확인)
  • Cube(최상위)의 Scale 은 (1, 1, 1) 이어야 한다. 이전 시간에 잘못 만들었으면 수정한다.
  • 대신 Cube > Visuals > Mesh 의 크기를 (0.1, 0.1, 0.1), Col 의 크기도 (0.1, 0.1, 0.1) 로 만든다.
  • 절대 Cube(최상위 오브젝트)의 크기를 변경하지 말 것! (최상위 스케일이 1이 아니면 스냅샷 포즈가 어긋난다)
  1. 재실행 스냅샷 — 크기를 바로잡았으면 다시 실행해서 핸드트래킹으로 잡는 손모양을 연출한 후 Hand Pose Recorder에서 Attach To Target (즉시) 버튼을 눌러본다.
  2. 저장 위치 확인 — 저장된 고스트 핸드는 Assets/HandPoses/Prefabs 에 저장된다.
  3. 프리팹 배치 — 에디터를 중지하고, 방금 스냅샷한 프리팹 (Assets/HandPoses/Prefabs/HandPose_Snapshot)을 끌어다가 Cube 자식으로 넣어준다.

4) 그립 포즈를 컨트롤러 잡기에 연결

  1. 모드 전환 — XR Origin 의 Hand Visualizer 를 비활성화하고, Left Controller 를 활성화한다.
  2. 그립 포즈 컴포넌트 — Left Controller 에 Controller Grab Hand Pose 컴포넌트를 부착한다.
  3. Cube 컴포넌트 — Cube 를 선택하고 Cube 컴포넌트를 부착한다.
  4. Cube Data 할당 — Cube 컴포넌트의 Cube Data 에 Assets/HandPoses/HandPose_Snapshot (ScriptableObject)를 끌어다가 할당한다.
  5. Grab Attach 생성 — Cube 자식에 있는 HandPose_Snapshot 을 선택하고 Tools > VR > Create Grab Attach... 를 선택하면, HandPose_Snapshot 자식으로 GrabAttach (Ghost Wrist) 가 생겨난다.
  6. Attach Transform 연결 — Cube 를 선택해서 XR Grab Interactable 컴포넌트의 Attach Transform 에 방금 생긴 GrabAttach (Ghost Wrist) 를 넣어준다.
  7. Interactor Attach 정렬 — Left Controller 를 선택하고 Tools > VR > Align Interactor Attach... 를 선택하면, Left_NearFarInteractor 자식으로 InteractorAttach (Wrist) 가 생성된다.
  8. 최종 확인 — 실행 후 큐브를 컨트롤러로 잡아보면 손 모델이 아까 스냅샷한 손모양 그대로 잡혀야 한다.
핵심 흐름 요약 
핸드트래킹으로 포즈 연출
   → Hand Pose Recorder "Attach To Target"으로 스냅샷
   → 스냅샷 프리팹을 Cube 자식으로
   → Cube 컴포넌트에 HandPose_Snapshot(SO) 할당
   → Create Grab Attach (Ghost Wrist) → XR Grab Interactable의 Attach Transform
   → Align Interactor Attach (Wrist)
   → 컨트롤러로 잡으면 스냅샷 손모양으로 잡힘
문제 해결 손이 어긋나게 잡히면 ① Cube 최상위 Scale이 (1,1,1)인지, ② Attach Transform이 GrabAttach (Ghost Wrist) 로 연결됐는지, ③ InteractorAttach (Wrist) 가 생성·정렬됐는지 확인하세요.
